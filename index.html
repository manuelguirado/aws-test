Aprendiendo AWS con un sitio web estático

Este es un ejemplo de sitio web estático que puedes subir a Amazon S3 para crear un sitio web hospedado en AWS.

Servicios de AWS para sitios web estáticos

Amazon S3

Almacenamiento de objetos escalable para alojar sitios web estáticos.

CloudFront

Red de entrega de contenido (CDN) para distribuir tu sitio web rápidamente.

Route 53

Servicio de DNS para configurar dominios personalizados.
